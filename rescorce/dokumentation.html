Hard-/Softwarevoraussetzung

Java ist nicht gerade superschnell. Das liegt an der Kombination aus Systemunabhängigkeit und eingebauten Internet-Sicherheitsvorkehrungen. Deshalb ist es sehr hilfreich, einen schnellen Computer zu haben. Wir empfehlen mindestens einen Pentium oder etwas Gleichwertiges für Windows und einen PowerMac für MacOS. Mit älteren Computern, zum Beispiel einem 486er oder einem Mac Quadra, geht's auch, aber natürlich langsamer. Alles, was noch langsamer ist, könnte zu langen Wartezeiten und einer Menge Frustration führen.

Benötigter Speicherplatz

Das zweite Kriterium hat damit zu tun, dass Java auf einer so genannten "virtuellen Maschine" läuft, einem simulierten Computer, der im Speicherinneren deines Computers untergebracht ist. Gewissermaßen wie ein Parasit, aber ein nützlicher. Das bedeutet, dass man viel Speicher (RAM) braucht, um Java problemlos laufen zu lassen. Man sollte mindestens 16 Megabyte RAM im Computer eingebaut haben, aber noch mehr wäre besser. Netscape 3.0 auf dem Macintosh ist ein besonders schlimmer Speicherverschwender.

Download und Installation unter Windows

Das Herunterladen und Installieren von Java ist einfach und kostenfrei. Es gibt verschiedene Möglichkeiten, Java für Windows zu erhalten

•Onlinedownload

•Offline-Download

Online: Bei der manuellen Installation wird eine ausführbare IFTW-Programmdatei (Install From The Web) heruntergeladen. Dieser Prozess erfordert kaum Benutzereingriff. Wenn Sie dieses Programm ausführen, werden alle erforderlichen Dateien aus dem Internet abgerufen. Daher muss die Internetverbindung während der Installation bestehen bleiben. Administratorrechte sind erforderlich, um Java unter Microsoft Windows zu installieren.

Treten bei der Onlinedownload-Option Probleme auf, laden Sie die Dateien mithilfe der Downloadoption "Offline" herunter.

Offline:  Für die Offlineinstallation müssen Sie eine ausführbare Datei herunterladen, die Sie auf der Seite für manuelle Java-Downloads finden. Hier stehen den Benutzern alle für die vollständige Installation benötigten Dateien zur Verfügung. Die Internetverbindung muss während der Installation nicht aufrechterhalten werden. Die Datei kann auch auf einen Computer ohne Internetverbindung kopiert und dort installiert werden.

Administratorrechte sind erforderlich, um Java unter Microsoft Windows zu installieren.

 

Download und Installation unter Linux

Es gibt zwei Möglichkeiten, Java für Linux zu erhalten

•RPM (RedHat Paket-Manager)

•Selbstextrahierende Binärdatei

Linux RPM verwendet RedHat Package Manager zur Installation von Java. Auf Ihrem Computer muss RPM verfügbar sein, damit Sie diese Methode verwenden können. Verwenden Sie andernfalls die selbstextrahierende Binärdatei.

 

Download und Installation unter Solaris

Es gibt verschiedene Möglichkeiten, Java für Solaris zu erhalten

•Solaris SPARC (32-Bit)

•Solaris SPARC (64-Bit)

Java wird als selbstextrahierende Binärdatei geliefert. Laden Sie das Paket herunter, das der Architektur und dem Betriebssystem Ihres Computers entspricht.

Spielanleitung

•Spielprinzip:

Das Spielfeld besteht aus zerstörbaren (blauen) und nicht zerstörbaren (roten) Steinblöcken.  Man muss Bomben legen, um sich einen  Weg durch das labyrintartig strukturierte Spielfeld zu schaffen.  Ziel ist es unversehrt  zu bleiben, den Gegner außer Gefecht zu setzten (Zwei-Spieler-Modus) oder den Zielblock zu erreichen (Ein-Spieler-Modus).

•Spielregel:

Es gibt 2 Arten von Steinblöcken. Sie sind durch unterschiedliche Farben kenntlich gemacht und beide sind undurchdringlich. Die einen sind aber, durch das Legen einer Bombe, sprengbar, die anderen  unzerstörbar. So muss man sich sowohl vor dem Gegner schützen (bei Zwei-Spieler-Modus) als auch das Ziel erreichen (Ein-Spieler-Modus).

Bomben können nur mit einer kleinen Verzögerung gelegt werden. Es sei denn die zweite Bombe wird unmittelbar neben der Ersten platziert. Dann addiert sich deren Sprengkraft und eine größere Explosion kommt zur Stande. Die Bombe hat einen Explosionsradius von je einem Feld nach links, rechts, oben und unten(Bombe selbst ist Mittelpunkt).

Jede Explosion kann einen töten; sei es die eigen oder die vom Gegner erzeugte. Deshalb muss man zwischen dem Bombermann und der Bombe sofort einen sicheren Abstand schaffen. Wenn man allerdings den Gegner in die Ecke getrieben hat und eine Bombe legen muss, die auch einen selbst treffen würde, stirbt nur der Gegner.

Es gibt zwei Bombermann. Also ist jeder des Anderen Feinds. Der rote Bombermann steht zur Beginn des Spiels oben links in der Ecke. Der Blaue unten rechts.  Sie können nicht durch den Gegner hindurch laufen und nur durch das Legen einer Bombe dem anderen schaden.

•Steuerung

Der rote Bombermann wird mit den Pfeiltasten gesteuert und mit der Enter-Taste legt er eine Bombe. Der Blaue wird mit W (oben)-A (links)-S (unten)-D (rechts)-Tasten gesteuert und legt mit der Space-Taste (Leer-Taste) eine Bombe.
